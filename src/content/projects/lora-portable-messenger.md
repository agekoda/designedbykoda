A rugged, feature-packed handheld messenger built around LoRa for long-range, infrastructure-free communication.

## Hardware

- ESP32-S3
- Battery management system
- 320×240 TFT display
- GPS module
- LoRa radio module
- Buzzer
- NeoPixel indicator
- Compass
- AHT20 temperature/humidity sensor

## Features

- Long-range portable messaging system
- Location-aware messaging (GPS)
- Heading/orientation via compass
