A large, modular RF analysis and interaction platform with multiple radios and a full UI — the flagship device in the lineup.

## Front

- 4” 480×320 TFT with resistive touch

## Core hardware

- ESP32-S3
- Dual NRF24L01 + PA/LNA (SMA antennas)
- CC1101 sub-GHz transceiver
- GPS module (external antenna)
- FM radio module (RDA5807M) + 3W amplifier
- IR TX/RX, buzzer, vibration motor
- SD card + RTC

## Power system

- 8420mAh dual-cell battery
- TPS63020 + INA226 + TP4056
- Dedicated battery PCB with pogo-pin interface

## Features

- Multi-protocol RF interaction
- Touch-based UI
- Audio output + FM radio
- Fully modular architecture with expansion capability
