A smart lighting system designed for both ambient display and Home Assistant integration.

## Hardware

- 8×8 NeoPixel matrix
- ESP32-C3 Supermini
- 2000mAh battery
- Frosted resin diffuser front
- TPS63020 ultra-low-power regulation

## Features

- WiFi-controlled RGB lighting
- Custom modes via Home Assistant
- Physical button control (mode + power)
- Deep sleep optimisation (~50µA idle)
